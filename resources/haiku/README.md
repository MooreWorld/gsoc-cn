# Haiku

## 基本介绍

[haiku](https://www.haiku-os.org) 是从 2007 年开始参与 GSoC 的一个操作系统项目，它的前身是 BeOS。

技术栈主要为 C/C++，系统底层相关项目比较多。

## 申请指南

社区会要求在其内部的 git 平台上提交 commit，相关指南在[这里](https://www.haiku-os.org/community/gsoc/2022/contributors)。
